
3-Controllerからtemplateにデータを渡す

◆コントローラーの編集◆

ディレクトリ
src/main/java/com/example/todo/controller/todo/TaskController.java

・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・

import org.springframework.ui.Model;

    @GetMapping("/tasks")
    public String list(Model model) {
        model.addAttribute("task", "Spring Bootを学ぶ");
        return "tasks/list";
    }

★コード解説★
addAttributeメソッド：第一引数を属性名、第二引数に表示させる値を入力する

◆テンプレートの編集◆

ディレクトリ
src/main/resources/templates/tasks/list.html

・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・・

<body>
<h1>タスク一覧</h1>
<a th:href="@{/}">トップページに戻る</a>

↓↓追記↓↓
<p th:text="${task}"></p>
</body>

★コード解説★
TaskController.javaで記述したaddAttributeのメソッドの第一引数を記述し、
第二引数で記述した内容を表示させている
